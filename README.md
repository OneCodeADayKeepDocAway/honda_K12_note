# 💡💡 K12 後煞車燈泡規格與更換 (Miniature light bulbs微型燈泡)

## 1. 燈泡功能與規格清單
K12 的後燈組主要分為「車身側」與「行李箱蓋側」兩部分。

| 燈泡位置,   | 功能說明, | 規格 (國際型號), | 備註 |
|----------|------|------|------|
|外側 (車身),|煞車燈 + 定位燈 (小燈),|7443 (大炸彈雙芯),|一顆燈泡有兩段亮度，開小燈微亮，踩煞車全亮。
|外側 (車身)|,方向燈,|7440 (大炸彈單芯),|琥珀色燈泡。
|內側 (車身),|定位燈 (小燈),|T10 (小炸彈),|僅在開啟小燈時恆亮。
|內側 (車身)|,倒車燈,|T15 (或 T10 通用),|白色燈光。

## 2. 重點區分：煞車燈 vs. 定位燈
很多車主會買錯規格，最關鍵的差別在於**「單芯」與「雙芯」**：

- 煞車燈 (7443)： 它是雙芯燈泡。底部有四根金屬絲接觸點。這是因為它必須同時負擔「小燈（低亮度）」與「煞車燈（高亮度）」兩種功能。如果您誤換成單芯的 7440，會發生踩煞車沒反應，或者燈泡亮度異常的情況。

- 輔助定位燈 (T10)： 位於行李箱蓋內側的那顆小燈。這只是單純的定位標示，規格跟您車內閱讀燈、前小燈常用的 T10 是一樣的。

## 3. 更換時的專業建議
- 直上 LED 的注意事項： 如果您想更換為 LED 燈泡，煞車燈請務必選擇**「帶有解碼或無極性」**的紅光 LED。若選擇不當，可能會導致儀表板噴出「VSA 或 ABS 故障燈」（因為 LED 電阻過低，電腦誤判燈泡燒毀）。

- 顏色選擇： 雖然燈殼是紅色的，但專家建議煞車燈仍選購**「紅光」**燈泡（或 LED），這樣透出來的色澤會比用「白光」直打更加飽滿深紅，質感更好。

- 清潔底座： K12 車齡較久，更換時建議檢查燈座是否有發黑或氧化跡象，可以用電子接點清潔劑噴一下，避免接觸不良。
